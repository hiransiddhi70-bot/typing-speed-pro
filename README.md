
# ⌨️ Typing Speed Pro

A fast, modern, and responsive Typing Speed Test application built using HTML5, CSS3, and JavaScript (Vanilla). This application helps users track and improve their typing speed and accuracy through an intuitive, interactive interface.

---

## ✨ Features

* **🚀 Live Words Per Minute (WPM):** Real-time tracking of your typing speed based on standard calculation formulas.
* **🎯 Accuracy Percentage:** Tracks your precision dynamically as you type each character.
* **❌ Mistake Counter:** Keeps a live count of incorrect keystrokes.
* **⚡ Characters Per Minute (CPM):** Measures your total correct characters processed per minute.
* **⏱️ 15s, 30s & 60s Timer:** Choose from multiple time modes to test your endurance and speed.
* **🏆 Best Score (Local Storage):** Automatically saves your highest WPM on your device so you can always challenge your high score.
* **📊 Test History:** Displays a list of your last 10 test attempts with WPM and accuracy metrics.
* **🌙 Dark Mode:** Easy-on-the-eyes dark theme toggle for late-night typing sessions.
* **📱 Mobile Responsive:** Clean UI layout that adapts seamlessly across mobile, tablet, and desktop screens.
* **🔄 Restart Test:** Instantly reset statistics and load a fresh random paragraph with a single click.
* **💾 Offline Support:** Runs entirely in the browser without requiring any database or backend server.

---

## 🛠️ Tech Stack

* **HTML5:** For semantic document structure and layouts.
* **CSS3:** Custom styling featuring dynamic CSS variables for fluid theme switching and responsiveness.
* **JavaScript (Vanilla):** Core application state logic, timing intervals, and event-driven data tracking.
* **FontAwesome:** For clean, modern UI dashboard icons.
