⌨️ Typing Speed Pro

A fast, modern, and responsive Typing Speed Test application built using HTML, CSS, and JavaScript.

✨ Features

- 🚀 Live Words Per Minute (WPM)
- 🎯 Accuracy Percentage
- ❌ Mistake Counter
- ⚡ Characters Per Minute (CPM)
- ⏱️ 15s, 30s & 60s Timer
- 🏆 Best Score (Local Storage)
- 📊 Test History
- 🌙 Dark Mode
- 📱 Mobile Responsive
- 🔄 Restart Test
- 💾 Offline Support

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
